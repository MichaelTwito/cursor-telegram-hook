Setup Guide

Get up and running in 3 steps.

Step 1 — Create a bot via BotFather
  1. Open Telegram and search for @BotFather
  2. Send /newbot
  3. Follow prompts — choose a name and username
  4. Copy the token and paste it in the Bot Pool tab
Step 2 — Connect your chat
  1. Open Telegram and find your new bot (search by username)
  2. Send /start to the bot
  3. Come back here and click Detect Chat ID on the bot card
  4. Send a test message to verify it works
Step 3 — Assign to a project
  1. Click Assign Project on the bot card
  2. Enter the absolute path to your project
  3. The hook will automatically use this bot when working in that project
  4. Unassigned bots act as the global fallback
Install hooks
# Claude Code stop hook (fires after every response)
python cursor-telegram-hook/install_claude_code.py install

# Telegram MCP tools (notify, ask, get_status mid-task)
python cursor-telegram-hook/telegram_mcp/install.py install
